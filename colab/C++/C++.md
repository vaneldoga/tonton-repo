# Uso da linguagem C++ no ensino da programação estruturada

#### Objetivo
Ensinar os conceitos sobre o **paradigma da programação estruturada**.

### O que é **programação estruturada**?  
É um paradigma (modelo) que organiza o código em **blocos lógicos bem definidos**, com o objetivo de tornar os programas mais **claros, legíveis e fáceis de manter.** Além disso serve como base para paradigmas mais avançados, como a **programação orientada a objetos**. 

Características principais:  
 - sequência: Instruções que são executadas uma após a outra, na ordem em que aparecem.
 - Decisão (seleção): Uso de estruturas como `if`, `else` e `switch` para tomar decisões.
 - Repetição (laços): Uso de `for`, `while` e `do-while` para repetir blocos de código.
 - Modularidade: Divisão do programa em funções ou procedimentos, cada um com uma responsabilidade clara.

#### Linguagens que suportam programação estruturada:
- C
- Pascal
- Python (também suporta outros paradigmas)
- Java (suporta programação estruturada, mas é mais voltada a orientação a objetos)
- C++ (suporta programação estruturada, mas é mais voltada a orientação a objetos)
